# Shared library for DataGalaxy Extensions (built with Typescript)

> This project host the shared library for all DataGalaxy extensions

## Prerequisite

-   [Rush](https://rushjs.io/)
-   [PNPM](https://pnpm.io/)

## Installation

```
rush update
```

## Install a new dependency

```
rush add <dependency>
```

## Build

```
pnpm build
```

## Watch for changes

```
pnpm watch
```

## Launch tests

```
pnpm test
```
